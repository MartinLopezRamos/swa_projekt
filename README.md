# 🌐 README – Webový Projekt

## 1. Téma webu
**Název projektu:** Moderní fitness a lifestyle platforma  
**Popis:**  
Webová aplikace zaměřená na fitness, zdravý životní styl a osobní rozvoj. Uživatelům poskytuje tréninkové plány, jídelníčky, edukativní články a nástroje pro sledování pokroku.

---

## 2. Mapa stránky (Site Map)

### 🏠 Home
- Úvodní přehled
- Nejnovější články
- Doporučené tréninky
- Call-to-action (registrace)

### 🧱 Architektura webu
- Frontend: HTML, CSS, JavaScript (React / Vue)
- Backend: Node.js / PHP
- Databáze: MongoDB / MySQL
- API: REST API pro komunikaci
- Hosting: Cloud (např. AWS / Vercel)

---

## 3. Podstránky (min. 10)

1. **Home** – Hlavní stránka
2. **About** – Informace o projektu
3. **Contact** – Kontaktní formulář
4. **Blog** – Články a novinky
5. **Workout Plans** – Tréninkové plány
6. **Nutrition** – Jídelníčky a výživa
7. **Progress Tracker** – Sledování pokroku
8. **User Profile** – Profil uživatele
9. **Login/Register** – Přihlášení a registrace
10. **FAQ** – Často kladené otázky
11. **Community** – Diskuze / fórum
12. **Pricing** – Cenové plány

---

## 4. Využití veřejně dostupných informací

Web využívá veřejně dostupné zdroje (open/public info):

- **Fitness data a doporučení**
  - WHO (doporučená fyzická aktivita)
  - Veřejné studie o výživě

- **API služby**
  - OpenFoodFacts API (nutriční hodnoty)
  - Google Maps API (lokace fitness center)

- **Obrázky a média**
  - Unsplash / Pexels (free obrázky)
  - Open-source ikony (např. Heroicons)

- **Technologie**
  - Open-source knihovny (React, Tailwind)
  - Dokumentace a veřejné repozitáře
