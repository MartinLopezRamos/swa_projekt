# Apex Boxing - Moderní Boxerský E-shop a Gym

Vítejte v projektu **Apex Boxing**, kompletním webovém řešení pro moderní boxerskou značku, showroom a gym. Tento projekt byl vytvořen s důrazem na minimalistický, vysoce kontrastní design ("Light Mode") a profesionální prezentaci prémiového vybavení.

## Struktura webu

Projekt se skládá z 11 plně propojených a responzivních stránek:

1.  **Domů (`index.html`)** - Hlavní vstupní stránka s nejdůležitějšími produkty a vizí značky.
2.  **Produkty (`sluzby.html`)** - Katalog e-shopu s možností filtrace kategorií, cenovým posuvníkem a gridem produktů.
3.  **Ceník (`cenik.html`)** - Přehledné tarify členství v gymu a ceny individuálních služeb.
4.  **O nás (`onas.html`)** - Historie značky Apex, naše mise a hodnoty.
5.  **Lektoři (`lektori.html`)** - Představení trenérského týmu s jejich specializací.
6.  **Galerie (`galerie.html`)** - Vizuální prezentace atmosféry gymu a detailů vybavení.
7.  **Blog (`blog.html`)** - Magazín s tipy pro trénink a údržbu vybavení.
8.  **Kontakt (`kontakt.html`)** - Kontaktní formulář, mapa a otevírací doba showroomu.
9.  **Kariéra (`kariera.html`)** - Nabídky práce pro trenéry a manažery.
10. **Obchodní podmínky (`obchodni-podminky.html`)** - Legislativní rámec nákupu.
11. **Ochrana údajů (`ochrana-udaju.html`)** - Informace o zpracování osobních údajů (GDPR).

## Technické požadavky a technologie

*   **HTML5** - Sémantické tagy (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`) pro optimální SEO a přístupnost.
*   **Javascript** - Dynamické prvky a logika Tailwind CSS.
*   **Tailwind CSS** - Moderní CSS framework využitý pro design systém (vlastní barvy, spacing a typografie).
*   **Google Fonts** - Využití písem **Montserrat** (nadpisy) a **Inter** (text) pro prémiový vzhled.
*   **Material Symbols** - Ikony od Google pro intuitivní navigaci.

## Designový systém

Web využívá jednotnou barevnou paletu:
*   **Primární:** Černá (`#000000`) - Elegance a autorita.
*   **Sekundární:** Červená (`#bc000c`) - Energie a vášeň pro sport.
*   **Pozadí:** Světle šedá (`#f9f9f9`) - Čistota a kontrast.

## Instalace a spuštění

Projekt nevyžaduje žádnou složitou instalaci. Stačí otevřít libovolný `.html` soubor v prohlížeči. Veškeré knihovny (Tailwind, Fonty) jsou načítány z CDN.
